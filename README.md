# 🌱 Smart Farm Website

A simple and responsive **Smart Farming website** developed using HTML.
This project presents the concept of modern agriculture and smart farming techniques through a clean and user-friendly interface.

---

## 🚀 Live Website

🔗 https://mrakesh8.github.io/Smart_Farm/

---

## 📌 Project Overview

The Smart Farm website is a **single-page static web application** designed to provide information about smart agriculture.
It delivers content in a simple, structured, and visually clear format, making it easy for users to understand modern farming concepts.

The entire website runs on the client side and does not require any backend services or database connectivity.
